04 — AI Chat
Chat That Takes Action
The LLM doesn't just respond — it uses tool calling to update the 3D body model in real time as you describe symptoms.

AI Tool Functions

update_muscle() Sets condition, pain, strength, mobility, and clinical summary for any muscle
add_knot() Records trigger points, adhesions, or spasms with severity
create_assessment() Generates clinical assessment with affected structures
search_knowledge_base() Queries RAG for evidence-based PT protocols

User Flow

  • "My left shoulder hurts when I raise my arm"
  • AI queries RAG for shoulder pathology context
  • AI calls update_muscle() on relevant rotator cuff muscles
  • 3D model updates in real time — affected muscles turn orange/red
  • AI responds with clinical reasoning and next steps
  • AI generates an evidence-based rehab workout

Patient Context

06 — Workout Generation
AI-Personalized Exercise Plans
Workout plans are synthesized from RAG clinical evidence and patient state. The AI avoids exercises for torn/strained muscles and adapts to available equipment.

Generation Pipeline

  • Patient Context: Current muscle states, pain levels, conditions from Convex
  • RAG Evidence: Retrieves relevant rehab protocols and exercise techniques
  • User Preferences: Goals, duration, available equipment, difficulty level
  • LangChain Synthesis: Prompt + LLM + JSON Parser → structured workout plan
  • Exercise Images: DALL-E 3 generates visual demonstrations for each exercise

Exercise Output

name: "Band Pull-Apart"
sets: 3
reps: 15
weight: "Light band"
target: ["rear_deltoid", "rhomboids"]
notes: "Focus on scapular retraction. Stop if shoulder pain exceeds 3/10."
reasoning: "Strengthens posterior chain to correct anterior tilt..."

09 — Differentiators
Why ChatPT Stands Out
🎯

Agentic AI, Not Just Chat

The LLM autonomously calls tools to update the body model as you describe symptoms. It's not a chatbot — it's an agent that takes clinical actions.

🔬

Evidence-Based RAG

Every recommendation is retrieved from real PT literature, not hallucinated. The pipeline uses agentic chunking and cross-encoder reranking for precision.

A 3D anatomical model with 100+ meshes gives both the user and the AI a shared spatial language for muscle groups and injury locations.
