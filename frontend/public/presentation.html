The Problem

Musculoskeletal care is broken

Describing pain is hard. Understanding anatomy is harder. Getting consistent, evidence-based guidance between PT visits is nearly impossible.

Vague descriptions

"My shoulder hurts somewhere" doesn't give practitioners enough to work with.

No continuity

Recovery happens between sessions, but patients lose track of progress and exercises.

Generic advice

YouTube exercises don't account for your specific condition, pain level, or recovery stage.

The Solution

Point. Speak. Heal.

ChatPT combines a real-time 3D anatomy model with an AI agent that listens, understands, and guides your recovery.

Click

Tap any muscle on the interactive 3D body to instantly select it, report how it feels, and set severity.

Speak

Describe your pain in natural language or voice. The AI identifies affected muscles and auto-highlights them.

Recover

Get personalized workout plans, track condition changes over time, and watch your body improve visually.

New Feature

Feelings-Based Muscle Input

Users describe how a muscle feels, not shader parameters. Select a feeling, set severity, confirm to save.

How it works

  • Each feeling maps to a clinical condition (e.g. Soreness → fatigued)
  • Severity slider (0–10) with green-to-accent gradient
  • Confirm button commits the change — no history spam
  • Strength decreases inversely with severity
  • Stiffness & Cramping also reduce mobility
  • Clear button resets to healthy instantly
  • 3D model updates reactively via Convex
Intelligence

Agentic AI Physiotherapist

Agent Tools

Capabilities

  • Parses natural language pain descriptions
  • Kinetic chain analysis & referred pain patterns
  • Asks clarifying questions (location, onset, character)
  • Research sub-agent on Cerebras Llama 3.3 70B
  • Streaming SSE with live progress indicators
  • Max 10 agent turns to prevent infinite loops
  • Full tool thread transparency

Voice Input

  • Browser-native recording via Web Audio API

  • Whisper transcription pipeline
  • Hands-free for injury scenarios

Features

  • Hover an exercise to highlight muscles on the model

  • Toggle active workout plan
